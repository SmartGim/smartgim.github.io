MAC 下高效写作方案

为啥要学会写作

为了更好的在互联网中分享自己的成果，写作在未来将会是一个人在社会上立足的基本能力，现在快餐式的学习环境下，产出有意义有价值的内容更是一项不可或缺的技能。所谓神器在手，天下我有，作为一个爱折腾的人，我将分享我在平时是怎样写作的。因为我常用的 Typora 最近更新了，所以以下我分享的组合是一个写作非常高效的方法。

神器组合

现在写作或者文档都是 Markdown ，在网络同步中最麻烦的是图片同步。所以我们比较需要一个图床工具。好用的组合如下：

Xnip + uPic + Typora

Typora 一定要是最新的，我的版本 0.9.9.32.1 (4191)

Xnip Version 2.1.1 (Build 57)

uPic 0.18.2 是当前的最新版本

如何配置

uPic 下如何配置请参考官方手册

我使用的是 Gitee 图床 和 SM.MS 注册版图床，注册之后获取到secret key

在 uPic 中进行配置

在 uPic 中设置输出格式为 Markdown 图床选择SMMS

Typora 配置

一定要是最新版的 Typora , 配置如下

这样在 Typora 中，如果是通过截图来插入图片的话，可以按如下操作

Command + Shift + X 调用 Xnip 的截图，截图完成后添加进剪切板，Command + V 直接插入，然后 Typora 中就会出现上传的选项，直接就可以上传，传完之后自动转化为 MarkDown 地址。

使用Typora 非常方便，写完就能看见效果 ，通过 Command + / 就能切换源代码模式。
